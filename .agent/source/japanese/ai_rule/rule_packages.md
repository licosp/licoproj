# 行動規範

プロジェクトの各パッケージ共通の行動規範。

---
## ルール

### コーディング規約
- **フレームワーク**: Vite + Vanilla JavaScript
- **スタイル**: CSS Variablesを使用したモダンでプレミアムなデザイン
- **デプロイ**: GitHub Pages (`docs` フォルダ構成)
- **ソースディレクトリ**: `src/` (メインロジックは `src/main.js`)

### プロジェクト固有のルール
- クライアントサイドのみで動作することを前提とする（サーバーサイド処理は行わない）。
