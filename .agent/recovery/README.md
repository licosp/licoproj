# .agent/.archive

このディレクトリは **緊急バックアップ・障害復旧用** のファイルを格納します。主な用途は次の通りです。

## 用途
- **クラッシュダンプ**: Lico が異常終了した際に自動生成されるメモリ・スタック情報。
- **緊急バックアップ**: 重要な操作（例: `git commit`、`git init`）直前に作成した一時的なスナップショット。
- **過去の設計資料**: `.agent/rules/README.md` 作成時に参照したメモやリファクタリング案など、将来的に参照したいが現在はアクティブに使わない資料。

## 保存方針
- ファイルは **削除せずにそのまま保存** します。必要に応じて `.agent/.internal/archive/` へ移動し、長期保存用の書庫に分類してください。
- ファイル名は **ISO8601/TIMESTAMP:Japan** 形式でタイムスタンプを付与し、時系列での検索を容易にします。

## 例
```
.agent/.archive/2025-11-27T12-21-53+09-00_git-commit.md
.agent/.archive/2025-11-27T00-00-00+09-00_refactor-rules-structure.md
```

## 注意点
- **外部公開しない**: これらのファイルは機密情報（パスやユーザー名）を含む可能性があります。
- **Git 管理外**: `.gitignore` に含めているため、リポジトリにコミットされません。
