## Historical Background

The **Shadow Repository** (`.agent/.internal/.shadow/`) was established on 2026-02-02 to manage files that require version control but should not be exposed to public hosting or the main repository's clean history.

However, its separation created a visibility outcome: contributions made to the Shadow Repository were invisible to the user and the main AI context, leading to "Phantom Work" where tasks were done but not credited. This document was created on 2026-02-11 to bridge that gap by projecting the Shadow's activities onto the Main's walls.
