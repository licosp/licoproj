# 質問ログ

## 目的
ユーザーから**重要な質問**を受けた際、`.agent/logs/questions/log_YYYY-MM-DD.md`にログを記録します。

## 重要な質問に含まれるもの
- アプローチやデザインに関するガイダンスのリクエスト
- 機能や制限に関する質問
- 新しいワークフローやシステムを作成するリクエスト
- 大きな変更につながる質問

## ログフォーマット
```markdown
### HH:MM:SS+TZ:TZ - [簡潔なタイトル]
**Q**: [ユーザーの質問]  
**Context**: [簡潔なコンテキスト]  
**Action**: [実行したこと]
```

## ログをスキップする場合
- シンプルな実装リクエスト
- 確立されたパターンに従った日常的なタスク
- 簡単な説明
