# 言語標準

## ファイルシステムの言語
- **ファイルシステム**: すべてのファイル（コード、ドキュメント、アーティファクト）は**英語**で記述する必要があります。
  - **例外**: ユーザーが明示的に他の言語を要求した場合。
  - **例外**: `~/.gemini/antigravity/brain/`内のシステムアーティファクト（例：`task.md`、`implementation_plan.md`）は**日本語**で記述する必要があります。

## プロジェクトディレクトリのファイル
- プロジェクトディレクトリに配置されるすべてのファイルは**英語**で記述する必要があります。
- すでに**日本語**で書かれた内容を保存する場合は、保存前に**英語に翻訳**してください。
- **例外**: 行動規範で明示的に定義された条件（例：ユーザーが日本語への翻訳を要求した場合）。

## コミュニケーション言語
- **チャット/通知**: **日本語**で応答します。
- **タスク更新**: タスク名、ステータス、サマリーは**日本語**で記述します。
