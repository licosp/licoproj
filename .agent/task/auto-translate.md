# 自動翻訳タスク（将来実装）

## 要件

特定のファイルを常に監視して、更新されたら自動的に翻訳し、結果を別のディレクトリに保存する常駐型エージェント。

## パス

- **監視対象**: `.agent/source/japanese/`
- **出力先**: `.agent/dist/`

## 実装候補

### 1. GitHub Actions（推奨）
- ファイル変更をトリガーに自動実行
- Gemini API経由で翻訳
- 自動コミット・プッシュ

### 2. Git hooks（pre-commit）
- コミット前に翻訳チェック
- 翻訳漏れを防止

### 3. VSCode拡張機能
- ファイル監視
- 保存時に自動翻訳

### 4. inotifywaitスクリプト
- Linux環境でのファイル監視
- バックグラウンド実行

## 制約

- Gemini Code Assistは常駐型エージェントとして動作不可
- セッションベースのため、外部ツールとの組み合わせが必要
