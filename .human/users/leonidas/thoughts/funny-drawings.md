## CLI版AIとの違い

```text
CLI 版                          IDE 版 (Antigravity)
─────────────────────           ─────────────────────────
gemini "質問1"                  [IDE プロセスが常駐]
  ↓ (プロセス終了)                    ↓
gemini "質問2"                  クエリ1 → API → レスポンス
  ↓ (プロセス終了)                    ↓
gemini "質問3"                  クエリ2 → API → レスポンス
                                      ↓
毎回新しいプロセス              IDE が文脈を保持し続ける
文脈は引数/ファイルで渡す       文脈は API のプロンプトに含まれる
```
